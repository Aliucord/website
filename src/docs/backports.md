## Built-in to Aliucord

- Viewing forwarded messages
- Upload size (new 10mb limit for non-nitro users)
- Display names
- Pomelo usernames (`@username` instead of `username#1234`)
- Polls
- Pronouns
- New profile badges (quests, developer, etc.)
- Avatar decorations
- Guild tags
- Nameplates
